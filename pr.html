Web Technologieën

Leer meer over de bouwstenen van het web: HTML, CSS, PHP en SQL.

HTML

Wat is HTML?

HTML (HyperText Markup Language) is de standaard opmaaktaal voor het maken van webpagina's. Het definieert de structuur en inhoud van een webpagina met behulp van tags.

  • Basis voor elke website.
  • Eenvoudig te leren en te gebruiken.
  • Werkt samen met CSS en JavaScript voor styling en interactiviteit.

CSS

Wat is CSS?

CSS (Cascading Style Sheets) wordt gebruikt om de stijl en lay-out van webpagina's te beschrijven. Het werkt samen met HTML om kleuren, lettertypen, marges en meer toe te passen.

  • Maakt websites visueel aantrekkelijk.
  • Scheidt inhoud (HTML) van stijl (CSS).
  • Ondersteunt responsive design voor verschillende apparaten.

PHP

Wat is PHP?

PHP (Hypertext Preprocessor) is een server-side scripttaal die wordt gebruikt om dynamische webpagina's te maken. Het wordt vaak gebruikt in combinatie met databases zoals MySQL.

  • Veel gebruikt voor webdevelopment.
  • Integreert gemakkelijk met HTML en databases.
  • Grote community en veel bronnen beschikbaar.

SQL

Wat is SQL?

SQL (Structured Query Language) is een taal voor het beheren en manipuleren van relationele databases. Het wordt gebruikt om gegevens op te slaan, op te vragen en te wijzigen.

  • Essentieel voor het werken met databases.
  • Gebruikt in combinatie met PHP voor dynamische websites.
  • Standaardtaal voor databasebeheer.
